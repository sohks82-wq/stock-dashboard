# Stock Dashboard

A responsive web application for displaying stock information, built with React, TypeScript, and Vite.

## Features

- Responsive design for desktop and mobile
- Display of stock symbols, names, prices, and percentage changes
- Clean, modern UI with hover effects

## Getting Started

### Prerequisites

- Node.js version 20.19+ or 22.12+ (due to Vite requirements)
- npm

### Installation

1. Clone the repository
2. Run `npm install` to install dependencies

### Development

Run `npm run dev` to start the development server.

### Build

Run `npm run build` to build the project for production.

## Project Structure

- `src/App.tsx`: Main application component
- `src/App.css`: Styles for the dashboard
- `src/main.tsx`: Entry point
- `src/index.css`: Global styles

## Technologies Used

- React
- TypeScript
- Vite
- ESLint
